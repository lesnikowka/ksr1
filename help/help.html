Жесткая система ДУ:

du1/dx = -500.005u1 + 499.995u2

du2/dx = 499.995u1 - 500.005u2

u1(0) = 7

u2(0) = 13

Реализованный численный метод:

Метод Рунге-Кутта 2-го порядка

x0, v0 = u0, xn+1 = xn + hn

vn+1 = vn + hn/2 (f(xn, vn) + f(xn + hn, vn + hn (f(xn, vn))))

n = 0, 1, 2…

Ограничение на шаг в связи с тем, что система жесткая

h < 2 / |макс. по модулю собственное число|

В нашем случае h < 2 / 1000

Необходимо численно решить данную систему ДУ.
